# Cookbook

Short focused examples showing how to accomplish specific tasks.

## Recipes

| Script | Description |
|--------|-------------|
| [animated_gif.tcl](gif-animation.md) | Play an animated GIF using frame cycling |
| [validate_user_input.tcl](validate-user-input.md) | Restrict entry fields to numbers or letters |
| [SVG widget patterns](svg-widget-patterns.md) | Rounded corners, theme-aware redraws, slide-in popups, DPI scaling |

## Running the recipes

```bash
./tclkit-9.0.3-Linux64-intel-tk cookbook/animated_gif.tcl
./tclkit-9.0.3-Linux64-intel-tk cookbook/validate_user_input.tcl
```
