# Gallery

The gallery contains fully functional demo applications ported from the Python ttkbootstrap gallery.

Run from the `ttkbootstrap-tcl-1.4.1` directory:

```bash
./tclkit-9.0.3-Linux64-intel-tk gallery/calculator.tcl
```

## Applications

| Script | Theme | Description |
|--------|-------|-------------|
| [calculator.tcl](calculator.md) | flatly | A functional calculator with colored numpad |
| [stopwatch.tcl](stopwatch.md) | cosmo | Start/pause/reset stopwatch |
| [equalizer.tcl](equalizer.md) | litera | Audio equalizer with draggable canvas sliders |
| [data_entry.tcl](dataentry.md) | flatly | Contact information form |
